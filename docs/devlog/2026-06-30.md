# 2026-06-30

## Apple Maps URL import, day-plan anchors, and trip updated dates

Added Apple Maps URL extraction alongside the existing Google Maps link import. The URL importer now accepts `maps.apple.com` links with `ll`, `q`, `address`, and Apple place-id variants such as `auid` / `place-id`, plus `/p/` short links that redirect to full Apple Maps URLs. Imported Apple candidates preserve `apple_maps_place_id` through review and database insert.

Persisted day-plan route anchors on `day_plans` via migration `0009_day_plan_anchor.sql`. Builder anchors are saved with the plan; saved-plan anchor changes post back immediately and survive reload. The anchor renders as the first row in the stop list, starts the Open directions route, participates in leg links, straight-line totals, driving distance persistence, route optimization, and weather lookup.

Updated the Trips list to show each trip's last updated date/time under the trip date range so duplicated trips are distinguishable.
