# 2026-06-30

## Apple Maps URL import, day-plan anchors, and trip updated dates

Added Apple Maps URL extraction alongside the existing Google Maps link import. The URL importer now accepts `maps.apple.com` links with `ll`, `q`, `address`, and Apple place-id variants such as `auid` / `place-id`, plus `/p/` short links that redirect to full Apple Maps URLs. Imported Apple candidates preserve `apple_maps_place_id` through review and database insert.

Persisted day-plan route anchors on `day_plans` via migration `0009_day_plan_anchor.sql`. Builder anchors are saved with the plan; saved-plan anchor changes post back immediately and survive reload. The anchor renders as the first row in the stop list, starts the Open directions route, participates in leg links, straight-line totals, driving distance persistence, route optimization, and weather lookup.

Updated the Trips list to show each trip's last updated date/time under the trip date range so duplicated trips are distinguishable.

## Trip list last saved timestamp

Reworked the Trips list timestamp semantics so `trips.updated_at` now means latest trip-owned save, not only edits to the trip shell row. Migration `0010_trip_activity_touch_triggers.sql` adds PostgreSQL triggers that touch the parent trip after inserts, updates, and deletes on itinerary items, packing lists/items, reservations, attachments, expenses, day plans, and day plan stops. The migration also backfills each trip from existing child-row timestamps where available. This keeps the Trips list fast because it still reads one indexed `trips` row per trip rather than aggregating child tables on every page load.

## Day-plan suggestions with anchors

Fixed the saved day-plan `Suggest stops` regression for plans that have a route anchor plus one real stop. The anchor now counts as a located route point in both the button enablement logic and the server-side suggestion centroid, while still requiring at least one saved stop. Browser QA on `http://127.0.0.1:5179/trips/1` used a one-stop anchored plan and verified the button opened the suggestions panel with internal and nearby discoveries.
