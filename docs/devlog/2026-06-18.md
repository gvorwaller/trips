## 19:?? EST - Print follow-up: tighter typography + spacing (f35748b)

### Summary
User feedback on the just-shipped print sheet: wanted a smaller font and
tighter spacing (the system print dialog gives no typography control), and
mused about rendering maps/docs as images. Tightened the print CSS; declined
the image work after weighing it. Shipped (`f35748b`), health green.

### What Was Done
- **Tighter print layout.** In the `@media print` block: shrank the print base
  from the 18px/1.5 screen scale to **9.5pt/1.25**, dropped headings
  (h1 20→16pt, h2 13→11pt, sub 11→9.5pt), reduced card/`@page` margins, and —
  the dominant fix — overrode the **44px touch tap-target `.line` height** to
  `min-height:0` + ~0.5pt padding (with `!important` to beat the component's
  scoped specificity), plus an 8pt `.meta` and an 11px print checkbox. Rows now
  sit on tight single lines; the test trip collapses from several airy pages
  toward a true one-pager.
- **Maps/docs as print images — assessed, declined.** Interactive Google maps
  don't print (blank tiles); the real fix would be a Static Maps `<img>`
  (needs the Maps Static API enabled in the gaylonphotos GCP project). Image
  attachments would be easy to embed; PDFs would need a client-side pdf.js
  rasterizer (most reservation docs are PDFs — highest value, most work). User
  chose to leave both as-is: the **reservation details already print**, which
  is the main benefit; the image idea was whimsical.

### Files Modified
- `src/app.css` — `@media print` typography/spacing tightening

### Key Technical Insight
The airy printout wasn't font size — it was the `.line { min-height: 44px }`
accessibility tap target bleeding into print, making every list row a half-inch
tall. Because that rule is component-scoped (Svelte hashes a class onto it,
raising specificity), a plain global print override loses; `!important` is
required to win. Verified by injecting the live stylesheet's `@media print`
rules as an on-screen `<style>` (faithful simulation minus `@page`) against the
test trip — rows compacted as expected.
